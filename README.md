## 目录说明
- 一个类别一个文件夹,文件夹里必含一个测试文件
- 可以使用go test测试
- 如array

```
 |- array
        |- array.go      //核心文件
        |- array_test.go //测试文件
        |- readme.md     //说明文档
```
## 包名
- 如果与golang包冲突的,包名加ext
- 例如:golang包time 命名exttime
